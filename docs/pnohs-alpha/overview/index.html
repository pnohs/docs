pnohs-alpha 概览

概览#

pnohs-alpha 是基于 pnohs 并行水文模拟出框架开发的并行水文模拟程序，集成多套水文产汇流模型。

特性#

pnohs-alpha 目前具有的功能如下:

  • 通过toml配置文件启动程序；
  • 基于pnohs库的大规模并行；
  • 基于pnohs库的多模型及模型参数的集成；
  • 参数率定
  • 集合预报
  • 模拟结果评价
